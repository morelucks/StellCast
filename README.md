# **StellCast**
 
Decentralized Prediction Markets on the Stellar Blockchain

Harnessing collective intelligence through fast, low-cost, trustless markets.

License: MIT
Built on: Stellar

---

## Vision

StellCast is a decentralized prediction market platform built on the Stellar blockchain, enabling users to trade on real-world event outcomes using fast, low-fee transactions.

By leveraging an automated market-making model (LMSR) and supporting Stellar assets such as USDC, XLM, and custom Soroban tokens, StellCast brings transparent, scalable, and accessible forecasting markets to a global audience.

---

## Why StellCast on Stellar?

Smart Contracts via Soroban

---

## Key Features

* LMSR automated market maker for continuous liquidity
* Multi-collateral support (USDC, XLM, custom tokens)
* On-chain settlement with no custodians
* Soroban smart contracts
* Tokenized outcome shares
* Instant redemptions

Winning outcome tokens are burned, and users receive collateral at a 1:1 redemption ratio.
